# How to Buy EmbryoLock

EmbryoLock is sold without accounts or emails.

## Supported chains
- Bitcoin (BTC)
- Ethereum (ETH)
- Polygon (MATIC)

## Purchase flow
1. Request a price:
   /pay/basic?chain=eth
   /pay/neon?chain=btc
   /pay/stealth?chain=poly

2. Send the exact amount shown to the displayed address.

3. Submit your transaction hash.

4. Receive a **one-time download link**.
   - Link expires
   - Single use only

## Rules
- No refunds
- No recovery
- Missed download windows require repurchase

This system is intentionally minimal.
