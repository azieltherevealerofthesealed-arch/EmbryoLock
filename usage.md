# Usage Guidance

EmbryoLock should only be used when the following assumptions hold:

---

## Appropriate Scenarios

- Sensitive data temporarily stored on a personal device
- Concern about theft after use
- Concern about delayed access
- Willingness to lose data rather than risk exposure

---

## Inappropriate Scenarios

- Long-term archival storage
- Shared systems
- Systems exposed to malware
- Any situation requiring recovery guarantees

---

## User Responsibility

- Choose a strong password
- Understand destruction behavior
- Accept that mistakes are irreversible

If accidental loss is unacceptable, do not use this tool.
